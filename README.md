## 📌 Overview

This project demonstrates a complete data analytics workflow — from raw data to a polished, presentable set of business insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, an interactive Power BI dashboard, and a final report and presentation.

**Goal:** Transform raw customer and sales data into insights that support better business decision-making.

---

## 🗃️ Dataset

Structured business dataset (3,900 rows × 18 columns) containing customer, product, sales, and transaction-level information.
